Introduction

Polar is Responsive Bootstrap 3 Environmental HTML Template built with Bootstrap 3.3.4, HTML5 and CSS3. Well Organized and valid Code.

Features

  • OnePage and Multipage
  • HTML5 and CSS3
  • Twitter Bootstrap 3.3.4
  • Fully Responsive Design
  • MailChimp Integrated
  • Working Contact Form
  • WordPress and Contact Form 7 Ready Markup
  • Google Fonts: Open Sans
  • 519 Font Icons (Font Awesome)
  • Clean & Minimal
  • Well Documented
  • Cross Browser Compatible - IE9+, Chrome, Firefox, Safari, Opera

JavaScript and Ajax

Polar imports following Javascript files.

  1. jQuery
  2. Custom scripts
  3. Google Map Api
  4. OWL Carousel
  5. Ajax Mailchimp
  6. etc

All code are beautifully written. You can customize easily. Just look the js code.
Example: If you want to Customize NewsLetter Subscription Url, just find the following block into the custom.js and change what you need:
 

/*

|----------------------------------------------------------------------------

| AJAX SUBSCRIBE

|----------------------------------------------------------------------------

*/

$("#subscribe").ajaxChimp({

    callback: mailchimpCallback,

    url: "http://codepassenger.us10.list-manage.com/subscribe/post?u=6b2e008d85f125cf2eb2b40e9&id=6083876991" // Replace your mailchimp post url inside double quote "".  

});
